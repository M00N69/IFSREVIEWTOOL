Aide - Interface Reviewer

Objectif : Relire le plan d'action, ajouter des commentaires pour l'auditeur.

Workflow :

  • Charger le package .ifsaudit reçu de l'auditeur.
  • Lire les corrections et preuves fournies par le Site.
  • Ajouter vos commentaires pour l'auditeur (non visibles par le Site).
  • Exporter le package avec vos commentaires et le renvoyer à l'auditeur.

Contraintes :

  • You can only modify your comments.
  • Each export creates a new version of the package (incremented number).
  • Make sure you always work on the latest version received.
  • Export the package after adding your comments.

Conseils :

  • Check the version number and date in the header before starting.
  • Keep a clear history of exchanged files (automatic naming).
  • Use your comments to guide the auditor in their decision.
  • Contact the auditor if in doubt.
